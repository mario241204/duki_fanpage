# 🎸 Duki Fanpage - Mini Proyecto Node.js

Bienvenido al repositorio de la **Fanpage de Duki**. Este es un proyecto desarrollado con Node.js utilizando el módulo nativo `http` para la gestión de rutas y servidores.

## 🔗 Acceso a la Web
Puedes ver el despliegue en vivo aquí:
👉 **[https://duki-fanpage.onrender.com/](https://duki-fanpage.onrender.com/)**

---

## ⚠️ Nota Importante
Este repositorio es un **mini proyecto educativo** que se realizó como práctica. 
> **Estado del proyecto:** No finalizado. 
> La estructura actual incluye la base del servidor, el sistema de rutas y los archivos estáticos principales, pero existen funcionalidades y secciones que no se llegaron a completar.

---

## 🛠️ Tecnologías utilizadas
* **Backend:** Node.js (Módulo HTTP nativo)
* **Frontend:** HTML5, CSS3 y JavaScript
* **Despliegue:** [Render](https://render.com/)

## 📂 Estructura del Proyecto
* `/modules`: Contiene la lógica del `router.js`.
* `/public`: Archivos estáticos (CSS, imágenes, JS de cliente).
* `/views`: Plantillas HTML.
* `server.js`: Punto de entrada de la aplicación configurado para entornos de producción.
